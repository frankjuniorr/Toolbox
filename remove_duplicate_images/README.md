Remove Duplicate Images
===========

## Description
  Esse script indentifica as imagens/vídeos duplicados do diretório "~/Dropbox/Camera Uploads".
  Uma vez identificado, o objetivo é remover desse diretório.

  O script funciona da seguinte forma:
  Ele cria um diretório temporário no `$HOME`, e recorta todos os arquivos duplicados do diretório "~/Dropbox/Camera Uploads" pra lá. Eu dou uma última conferida no olho mesmo e deleto por conta própria

## Use
```bash
./remove_duplicate_images.sh
```
