Mickey Feio
===========

## Description:
Script auxiliar para fazer o download de todas as imagens do Instagram do projeto "Mickey Feio" usando a hastag `#mickeyfeio2019`

## Uso:
```bash
pip3 install -r requirements.txt
python3 mickey_feio.py
```
